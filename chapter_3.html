Programming Logic

Programming logic serves as the backbone of computer programming, just as a recipe tells you step-by-step how to make a dish, programming logic guides a computer through a series of instructions to perform a task.

In programming, logic constructs such as conditional statements (if-else), loops (for, while), variables (placeholders for data), and functions that provide the framework for organizing and manipulating data.

It is the foundation to solve any given problem when developing a software. It precedes the use of any specific programming language, laying the groundwork for effective problem-solving and algorithmic thinking.

What are Variables?

Something that can store information, be referenced and manipulated. Each variable has a type, a name and a piece of information stored inside of it (just like a cardboard box);

Types of Variables

There are many types of variables, but we will be focusing on primitive type variables, and those are:

  1. Integer

    a variable that can store an integer number. Can't and will not hold any decimal values

  2. Boolean

    can only store a value of either TRUE or FALSE

  3. Floats

    floating point data types, it is able to store numbers with decimal places

  4. Double

    very similar to the variable Floats. It is also a floating point data type, it is able to store numbers with decimal places. Can store up to 64 bits of information

  5. Strings

    can store strings of letters (words and sentences). Useful for displaying text and storing input information (information the user puts into our program)

  6. Char

    char stands for character. Each char variable holds only one character - "*" "!" "R" ... Useful when a programmer wants to read one button of one character in a string without using a String variable.

Sometimes you're going to want to keep track of things such as a user's name or score, and that's why variables are so useful. By creating a variable you can store this information into it and then reference, add to or modify it. When naming variables in programming, there are some common rules adn conventions that are typically followed across various programming languages. These rules help improve code readability and maintainability. Here are some general guidelines:

  1. Use meaningful names

    choose variable names that accurately describe the data they hold or the purpose they serve in your code. This makes your code easier to understand for yourself and others who may read it later

  2. camelCase convention

    Most programmers name variables according to camelCase. Don't capitalize the first word, but capitalize the first letter of all words after it (e.g., nameThisVariable)

  3. Start with a letter or underscore

    variable names should begin with a letter (a-z, A-Z) or an underscore (_). They cannot start with a number

  4. Avoid reserved words

    don't use keywords or reserved words of the programming language you're working with as variable names

Following these guidelines while naming variables can contribute significantly to enhancing the clarity and organization of your codebase. By choosing meaningful and descriptive names, following established conventions, and maintaining consistency throughout your code, you make it easier for yourself and other developers to understand, modify, and maintain the code over time.

What are Loops?

Loops are control structures that allow you to execute a block of code repeatedly based on a specified condition. They are used when you want to perform the same or similar tasks multiple times without having to duplicate code.

Effective use of loops requires understanding the problem at hand and devising a strategy to iterate over data or execute certain actions multiple times until a specific condition is met. This ability to control the flow of execution through repetition is a fundamental aspect of programming logic.

There are three main types of loops:

  1. For loop

    used when you know exactly how many times you want to execute a block of code. It iterates over a sequence (such as a range of numbers) and executes the block of code for each item in the sequence

  2. While loop

    used when you want to execute a block of code as long as a specified condition is true. It keeps looping until the condition becomes false

  3. Do-while loop

    similar to a while loop, but the block of code is executed at least once before the condition is checked (not available in all programming languages)

...

In summary, loops are fundamental to many programming tasks, such as iterating over elements in a list or performing repetitive calculations. They help in writing more concise and efficient code.

What are Functions?

A function is a self-contained block of code that performs a specific task or set of tasks. Functions provide a way to modularize code by encapsulating a series of instructions that can be reused throughout a program. They allow you to break down complex problems into smaller, more manageable parts, making your code more organized, readable, and maintainable.

Functions typically take input parameters (arguments) and may produce output (return value), although they can also perform actions without returning a value. They can be called (invoked) from various parts of a program to execute the code contained within them.
