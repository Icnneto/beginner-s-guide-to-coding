How do we Write Code?

We use IDE's (Integrated Development Environments) to write code. IDE's are places to write, run and debug code and also convert it to machine code. They're like any other program in your computer. For that, we need to understand that each language has it's own set of rules you must follow within an IDE.

Learning a computer language is very similar to learning a real language. All programming languages has a set of rules you must follow when writing code in that language. In computer science it is called Syntax.

Syntax

Rules you must follow to a tee if you want your program to run correctly. Each programming language has their unique syntax. Therefore, for each one of them, you must study:

  1. how you type out certain functions

  2. what you put at the end of each line of code

  3. how you set up certain functions
